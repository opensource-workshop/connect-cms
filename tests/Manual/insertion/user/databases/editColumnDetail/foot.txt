    <p><b>タイトルの設定</b></p>
    <ul>
        <li class="card-text">新着情報等のタイトル表示に使う項目を、データベース毎に１つ指定できます。</li>
    </ul>
    <p><b>本文の設定</b></p>
    <ul>
        <li class="card-text">新着情報等の本文表示に使う項目を、データベース毎に１つ指定できます</li>
    </ul>
    <p><b>イメージの設定</b></p>
    <ul>
        <li class="card-text">新着情報等のイメージ表示に使う項目を、データベース毎に１つ指定できます</li>
    </ul>
    <p><b>選択肢の設定</b></p>
    <ul>
        <li class="card-text">選択肢の追加、削除、表示順の変更ができます。</li>
    </ul>
    <p><b>チェック処理の設定</b></p>
    <p>「１行文字列型」、「複数行文字列型」について、下記の設定ができます。</p>
    <ul>
        <li class="card-text">数値のみ許容</li>
        <li class="card-text">英数値のみ許容</li>
        <li class="card-text">入力最大桁数 ※数値用</li>
        <li class="card-text">入力最大文字数</li>
        <li class="card-text">最大値</li>
        <li class="card-text">最小値</li>
    </ul>
    <p>「日付型」について、下記の設定ができます。</p>
    <ul>
        <li class="card-text">指定日数以降の日付のみ許容</li>
    </ul>
    <p><b>項目毎の固有設定</b></p>
    <ul>
        <li class="card-text">「時間型」で、分刻みの設定ができます。</li>
    </ul>
    <p><b>キャプションの設定</b></p>
    <ul>
        <li class="card-text">キャプション内容の追加、キャプション文字色の変更ができます。</li>
    </ul>
    <p><b>DBカラム設定</b></p>
    <ul>
        <li class="card-text">一覧・詳細画面への表示・非表示を設定できます。</li>
        <li class="card-text">権限毎に一覧・詳細画面または、登録・編集画面で表示カラムの表示・非表示を設定できます。</li>
        <li class="card-text">並べ替え項目・検索対象項目として指定できます。</li>
        <li class="card-text">行グループ・列グループを設定して、表示カラムを横並びに表示する事ができます。</li>
